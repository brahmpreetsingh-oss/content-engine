Create channel-specific distribution assets from approved, final content.

$ARGUMENTS should contain either: the approved final content (pasted in full), OR a reference to an approved story in the pipeline.

## STEP 1: Confirm Content is Approved

Before creating any assets, confirm with the user that the content provided is the final, marketing-team-approved version — not a draft.

If the user indicates it is a draft, ask them to confirm editorial approval before proceeding.

## STEP 2: Read and Classify the Content

From the provided content, identify:
- Content Tier (Tier 1 = long-form blog / Tier 2 = focused asset / Tier 3 = social)
- Primary target persona (Executive / Technical / Delivery / Marketing)
- Platform focus (if any: HubSpot / Drupal / Acquia / AWS)
- Industry vertical (if any)
- Core message in one sentence

## STEP 3: Run the Distribution Agent

Use the distribution-agent to produce all appropriate assets for the identified tier:

**Tier 1 assets**: LinkedIn post (primary + alternative) · Newsletter blurb · Sales enablement summary · Campaign snippet · Internal win announcement · Platform POC brief (if platform-focused) · Full distribution sequence plan · Creative brief for visual assets

**Tier 2 assets**: LinkedIn post · Newsletter blurb · Campaign snippet · Focused activation sequence

**Tier 3 assets**: LinkedIn post (2 variants) · Hashtag recommendations

## STEP 4: Social Format Recommendation

For any LinkedIn content produced, also recommend the optimal content format using the Social Post Understanding framework:

- **Video/write-up**: For stories with emotional quotient or specific team members
- **Carousel**: For process-driven or technical stories (each slide = one aspect)
- **Infographic**: For technical/diagrammatic content
- **Long-form with abstract image**: For pure thought leadership angles

Provide a brief for the Creative plugin if a visual asset is needed.

## STEP 5: Present Distribution Pack

Deliver all assets clearly labeled and ready to copy/use. Include:
- A recommended publish sequence with suggested timing
- Notes on which assets need creative assets before publishing (flag for Creative plugin)
- Any platform-specific guidance (e.g., best time to post on LinkedIn for B2B)

## Rules
- Never create assets from unapproved drafts
- Keep all channel assets consistent in message, not identical in language
- Social posts must follow the Social Post Understanding framework: scroll-stop hook · value-adding body · right creative format
- Flag any asset that references client-identifiable information for review
